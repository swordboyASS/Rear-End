:star: 揭开服务器的面纱

先引用一段网络上对于服务器的理解 from wikipedia
```
服务器（英语：Server）指：

* 一个管理资源并为用户提供服务的计算机软件，通常分为文件服务器（能使用户在其它计算机访问文件），数据库服务器和应用程序服务器。

* 运行以上软件的计算机，或称为网络主机（Host）。
```

通俗且不专业的说，服务器就是更高配置（CUP,显卡（集成）,芯片组,内存,磁盘系统、网络）的固定电脑,并且做到24小时不能关闭。普通电脑根本做不到这样的程度
